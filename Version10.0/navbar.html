How to Create a Navbar

This guide will show you how to create different types of navigation bars (navbars) using HTML and CSS. Example code is provided below, along with explanations.

Example 1: Basic Navbar

Here is the code to create a basic horizontal navbar:

<div class="navbar"> <a href="#home">Home</a> <a href="#about">About</a> <a href="#services">Services</a> <a href="#contact">Contact</a> </div>

Add the following CSS to style your navbar:

.navbar { background-color: #333; overflow: hidden; } .navbar a { float: left; display: block; color: white; text-align: center; padding: 14px 16px; text-decoration: none; } .navbar a:hover { background-color: #ddd; color: black; }

Explanation:

  • The .navbar class creates a container with a background color and ensures the contents don’t overflow.
  • Each a tag represents a link and is styled with padding, text color, and a hover effect.

Example 2: Navbar with Right-Aligned Links

You can align some of the links to the right by using an additional class:

<div class="navbar"> <a href="#home">Home</a> <a href="#about">About</a> <div class="navbar-right"> <a href="#login">Login</a> <a href="#signup">Sign Up</a> </div> </div>

CSS:

.navbar-right { float: right; }

Explanation:

  • The .navbar-right class moves the links inside it to the right by setting float: right;.

Example 3: Responsive Navbar

For a responsive navbar, use media queries:

<style> .navbar a { float: none; width: 100%; text-align: left; } @media (min-width: 600px) { .navbar a { float: left; width: auto; } } </style>

Explanation:

  • On smaller screens, links stack vertically by setting float: none; and width: 100%;.
  • On wider screens (min-width: 600px), links return to their horizontal layout.
